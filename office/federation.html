How to Join the Federation

1
Install maw on your machine
git clone https://github.com/Soul-Brews-Studio/maw-js
cd maw-js && git checkout alpha
bun install && bun link
2
Configure your node
{
  "node": "your-hostname",
  "port": 3456,
  "peers": ["http://white.wg:3456"],
  "namedPeers": [
    { "name": "white", "url": "http://white.wg:3456" }
  ],
  "agents": {
    "your-oracle": "your-hostname",
    "neo": "white"
  },
  "mqtt": {
    "broker": "ws://white.wg:9001",
    "selfName": "your-oracle",
    "selfHost": "your-hostname"
  }
}
Save to ~/.config/maw/maw.config.json
3
Start the server
pm2 start src/server.ts --name maw --interpreter bun
4
Test federation
# Send to a remote agent
maw hey white:neo-oracle "hello from your-hostname"

# Check transport status
maw transport status

Transport Priority

1. tmux — local, 0ms (same machine)
2. mqtt — cross-internet, real-time (WebSocket broker)
3. http — WireGuard/LAN, ~6s (curlFetch)
4. lora — off-grid, future (Meshtastic hardware)
