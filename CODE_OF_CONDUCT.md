# Code of Conduct

## Standard

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be kind. Assume good faith.
- Name the behavior, not the person.
- Disagreement is welcome. Hostility is not.
- This project ships fast and breaks things. Be patient with churn.

## Reporting

For Code of Conduct concerns, email **conduct@soulbrews.studio** or DM a maintainer.

Reports are handled privately. Retaliation against reporters is itself a violation.

## Scope

Applies to all project spaces — issues, PRs, discussions, commit messages, and any synchronous channels linked from this repo.
